## 📖 Overview
This repository contains the solutions for Lab 11, which focuses on handling binary data in Java. The project demonstrates how to read and write primitive data types using binary streams, how to encapsulate that data into objects, sort it, and finally, how to serialize those objects directly to a binary file.

## 🚀 Features & Problem Breakdown

### Problem 1: Basic Binary I/O (`Lab11Prob01.java`)
* **Goal:** Read an existing binary file (`person.dat`) and create an exact duplicate.
* **Implementation:** Uses `DataInputStream` and `DataOutputStream` wrapped around `FileInputStream` and `FileOutputStream`. It loops continuously until an `EOFException` is caught, reading `int`, `UTF` strings, and `double` values in a specific sequence and writing them to `people-copy.dat`.

### Problem 2: Sorting and Data Processing (`Lab11Prob02.java` & `Person.java`)
* **Goal:** Read the binary data, convert it into Java objects, sort them by salary, and output the formatted strings to a new file.
* **Implementation:** * Introduces the `Person` model class, which implements the `Comparable<Person>` interface to sort instances by salary in descending order.
    * Reads `person.dat`, populates an `ArrayList<Person>`, sorts the list using `Collections.sort()`, and writes the currency-formatted `toString()` representation of each person to `people-salary-sorted.dat`.

### Problem 3: Object Serialization (`Lab11Prob03.java`)
* **Goal:** Write actual Java object instances to a binary file rather than plain text or primitive binary data.
* **Implementation:** * Updates the `Person` class to implement the `Serializable` interface.
    * Replaces the `DataOutputStream` from Problem 2 with an `ObjectOutputStream`, writing the fully instantiated and sorted `Person` objects directly to `people-salary-sorted-objects.dat` using `writeObject()`.

## 🛠️ Technologies Used
* **Java 8+**
* **File I/O:** `File`, `FileInputStream`, `FileOutputStream`, `DataInputStream`, `DataOutputStream`, `ObjectOutputStream`
* **Collections Framework:** `ArrayList`, `Collections.sort()`
* **Interfaces:** `Comparable<E>`, `Serializable`
